# 🌾 Projeto Agrinho 2026 — Inovação e Sustentabilidade no Campo

Este é um projeto de website responsivo desenvolvido para o **Programa Agrinho 2026**. O objetivo do site é divulgar as ideias, pilares e propostas de inovação tecnológica e sustentabilidade voltadas para o ambiente rural, destacando a participação do aluno e desenvolvedor **Victor Kovaleski**.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias web fundamentais, sem a necessidade de frameworks externos, garantindo leveza e rapidez no carregamento:

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Design moderno, paleta de cores customizada (Mobile-First) e layout responsivo.
* **JavaScript (Vanilla):** Interatividade, navegação com scroll suave e animações dinâmicas de surgimento (*fade-in*) ao rolar a página.
